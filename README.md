# David.Eadie.net

## Run

To start the app locally run:

```
npm run dev
```

## Test

All pushes to branches will trigger a release to [the staging area](https://david-test.eadie.net). The latest push will
win.

## Release

All pushes to `main` will automatically deploy the site.
